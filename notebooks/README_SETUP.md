## рџ“Љ What Each Notebook Produces

### Notebook 1 вЂ” EDA
- Missing value analysis and forward-fill imputation
- Distribution plot of Global Active Power
- Full time-series plot (daily average, 4 years)
- Seasonal monthly trends by year
- Hourly pattern вЂ” weekday vs weekend
- Weekly pattern by day of week
- Seasonal boxplots (Winter / Spring / Summer / Autumn)
- Sub-meter energy share pie chart
- Feature correlation heatmap
- Outlier detection (Z-score and IQR)
- Saved cleaned datasets for modeling

### Notebook 2 вЂ” Modelling
- Lag features (1h, 2h, 3h, 6h, 12h, 24h, 48h, 168h)
- Rolling mean & std features (3h, 6h, 24h windows)
- Cyclical time encoding (sin/cos for hour, day, month)
- **4 models trained and compared:**
  - NaГЇve persistence baseline
  - Ridge Regression
  - Random Forest (200 trees)
  - XGBoost (500 boosting rounds)
  - LSTM (optional, requires TensorFlow)
- Metrics: RMSE, MAE, MAPE, RВІ
- Model comparison bar charts
- Feature importance chart
- Residual analysis
- Walk-forward time-series cross-validation (5 folds)
- All models saved to `models/`

## рџ›‘ Troubleshooting

| Problem | Fix |
|---------|-----|
| `FileNotFoundError: household_power_consumption.csv` | Place the CSV in `data/` (Step 1) |
| `ModuleNotFoundError: xgboost` | `pip install xgboost` |
| `ModuleNotFoundError: tensorflow` | `pip install tensorflow` вЂ” or just skip; the LSTM cell auto-skips |
| Notebook 2 fails on hourly_clean.csv | Run Notebook 1 first to completion |
| Memory error on large dataset | Use `df.sample(500_000, random_state=42)` for plotting cells |
